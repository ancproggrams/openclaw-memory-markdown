# Publish checklist

## Before first public release

- [ ] Replace placeholder repository URL in `package.json`
- [ ] Replace placeholder homepage URL in `package.json`
- [ ] Replace placeholder bugs URL in `package.json`
- [ ] Confirm plugin name is final
- [ ] Confirm `openclaw.plugin.json` metadata is final
- [ ] Run `npm test`
- [ ] Run `npm pack --dry-run`
- [ ] Install once in a clean OpenClaw environment
- [ ] Verify README examples
- [ ] Add repo screenshots if desired

## Optional but recommended

- [ ] Add CI workflow
- [ ] Add semantic version tag strategy
- [ ] Add CONTRIBUTING.md
- [ ] Add issue templates
- [ ] Add ClawHub-specific publishing notes once registry path is confirmed
